# 📄 Sistema de Recibos (DevDesign Manager)

Um sistema web completo, moderno e responsivo para a gestão e emissão de recibos, desenhado com foco na experiência do usuário e eficiência administrativa. O sistema é voltado para agilizar processos financeiros e de controle de equipes (squads).

## ✨ Funcionalidades Principais

- **Emissão Rápida:** Geração de recibos de forma ágil e intuitiva.
- **Gestão de Squads:** Controle e visualização das equipes diretamente pela interface.
- **Visual Profissional:** Interface de usuário construída utilizando Tailwind CSS com design premium.
- **Dashboard Integrado:** Visualização de dados de forma gráfica utilizando Chart.js.
- **Design Responsivo:** Adaptado para funcionar perfeitamente em dispositivos móveis e desktops.
- **Versão para Impressão:** Geração de recibos prontos para impressão ou salvamento em PDF.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **Tailwind CSS:** Estilização moderna, responsiva e baseada em utilitários.
- **JavaScript (Vanilla):** Lógica do sistema e interações dinâmicas.
- **Chart.js:** Biblioteca robusta para a renderização de gráficos.
- **Vercel:** Plataforma escolhida para o deploy contínuo e rápido do sistema.

## 🛠️ Como Executar Localmente

Para rodar este projeto na sua máquina, não é necessária uma configuração complexa com banco de dados, pois a aplicação roda no lado do cliente.

1. **Clone este repositório:**
   ```bash
   git clone https://github.com/vytor0-IFPI/meu_sistema_de_recibos.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd meu_sistema_de_recibos
   ```

3. **Execute o sistema:**
   Basta abrir o arquivo `index.html` no seu navegador de preferência, ou utilizar a extensão **Live Server** no VS Code para atualizar automaticamente enquanto edita o código.
